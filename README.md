# Usage
**Note it is still in the development stage it wouldn't be stabled**

Follow the steps below to check out this project:
* ```git clone https://github.com/SoarPro/SoarPro.git```
* ```cd SoarPro```
* ```python soarshell.py```
* You can optionally convert it to a binary file with pyinstaller module
* ```pip install pyinstaller```
* ```pyinstaller --onefile soarshell.py```
* ```cd dist```
* ```./soarshell```
